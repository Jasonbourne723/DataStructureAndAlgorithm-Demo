# DataStructureAndAlgorithm-Demo

掌握了数据结构与算法，看待问题的深度，解决问题的角度就会完全不一样。因为这样的你，就像是站在巨人的肩膀上，拿着生存利器行走世界。数据结构与算法，会为你的编程之路，甚至人生之路打开一扇通往新世界的大门。
学习算法与数据结构对我们的帮助：

- 建立时间复杂度、空间复杂度意识
- 写出高质量的代码
- 能够设计基础架构
- 提升编程技能
- 训练逻辑思维。
